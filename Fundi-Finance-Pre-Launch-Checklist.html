Install Bubblewrap CLI, run bubblewrap init --manifest https://fundiapp.co.za/manifest.json, configure assetlinks.json on the domain. This wraps the PWA as a native Android app. No code rewrite needed.
Blocker

This file links the Play Store app to your domain — without it, the TWA shows a browser address bar. Add a Next.js route that serves the JSON. You'll need the SHA-256 fingerprint of your signing key.

Go to play.google.com/console. You need a Google account and to pay the $25 registration fee. The account can be personal or business.
Blocker

Apple's App Store has stricter PWA policies. Option A: Use Capacitor (ionic-team/capacitor) to wrap the web app as a native iOS app — easiest path. Option B: Submit as a web-based app using WKWebView. Decision needed before starting iOS work.
Important

Required to distribute on the App Store. Apply at developer.apple.com. Processing can take up to 48 hours for new accounts.
