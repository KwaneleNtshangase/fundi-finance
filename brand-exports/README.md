## Why not just make everything transparent

Because platforms actively disagree, and each rejects the other's format:

- **Meta rejects a white background.** It wants alpha. (This is what bounced.)
- **Apple rejects alpha.** A transparent iOS icon renders **black** on the home
  screen. Opaque or nothing.
- **PWA maskable needs opaque.** The mask crops to a device-chosen shape; alpha
  inside the safe zone shows the user's wallpaper through your icon.
- **Open Graph wants opaque.** Several clients composite alpha onto black, so a
  navy wordmark on "transparent" becomes navy on black — invisible.
- **favicon wants transparent**, so it sits cleanly on any browser chrome.

So both versions exist and the folder name tells you which is which. The
generator asserts it too: every file is checked after writing, and the build
fails if a file in `transparent/` turns out opaque or vice versa.

## Flat vs gradient

`icon/` uses the **flat** mark. `icon-gradient/` has the gradient one at 1024
and 512 only.

The gradient's mid-stop puts a muddy dark streak through the teal stroke, which
is invisible at poster size and obvious at 64px. Anything icon-sized should be
flat. The large lockups in `lockup/` keep the gradient, where it reads as depth
rather than dirt.

## The one rule that's easy to break

**The icon is the letter N**, so the wordmark is `OTHO`, not `NOTHO`. Setting
the mark next to the full word reads *N-NOTHO*. If you need the name as plain
text with no mark, type the whole word — but never put the whole word beside the
mark. Everything in `lockup/` is already correct; this only bites when someone
assembles a new lockup by hand.

## Colour

| | Hex | Contrast on white |
|---|---|---|
| Teal, bright | `#01A0AA` | 3.18:1 — graphics only, never text |
| Teal, deep | `#007A85` | 5.09:1 — AA for text and buttons |
| Gold | `#EFB343` | 1.88:1 — decorative only, fails even 3:1 |
| Navy | `#0D368D` | 10.88:1 — headings, and text on gold |

Gold can't carry meaning by itself on a light surface. Where a gold surface
needs a label, put navy on it (5.80:1).
